# These are the commans we have learnt today

1. To clone a branch, use:
   git clone
   e.g. git clone https://github.com/collinsefe/cloud-skills-infra.git

2. to create a new branch, use:
   git checkout -b
   e.g. git checkout -b april-2025

3. to add new changes, use:
   git add 
   e.g. git add new-file.txt

4. to commit the changes you have made, use:
   git commit 
   e.g. git commit -m "create new file"

5. to check the status of your files, use:
   git status

6. to push your changes to github, use:
   git push 
   e.g. git push origin april-2024

7. to change back to main branch, use:
   git checkout main 

8. to uodate your local branch with the origin, use:
   git pull
   
8. git log

9. git diff
